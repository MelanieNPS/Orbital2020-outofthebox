Rules

Classic Sudoku

In a classic Sudoku puzzle, there are 9 rows, 9 columns and 9 boxes

Each row, column and box must contain the numbers 1 - 9

The same number cannot appear twice in a row, column or box

Hyper Sudoku

The rules for hyper sudoku are similar to that of classic sudoku except that there are now 4 extra boxes

These 4 new boxes (dark blue color) must also contain the numbers 1 - 9 with no repeats

Controls

Click Mode

Clickmode must be active to use

* To input a number into a cell, click on the number from the number panel then on the cell

* To erase, click on the erase button in the bottom panel, then click on the cell

Keyboard

* Cells can be navigated with arrow keys

* Numbers can be inputted using number keys (1 - 9)

Keyboard shortcuts can only be used when an input cell is highlighted

* Toggle autocheck on & off, press "a"

* Toggle clickmode on & off, press "c"

* Fill notes for empty cells, press "f "

* Toggle notes on, press "n"

* Toggle notes off, press "m"

Refer to Options -> Display Keyboard Shortcuts on main page

Tools

Check

Cells turn green if numbers are right, turns red if numbers are wrong

Auto-check

Checks if number input is valid, according to the rules

Numbers will turn red if rules are broken

Clear All

Resets the current puzzle

Notes/ Fill Notes

Allows user to input notes into the cell

Controls for the notes are the same

Fill notes button will automatically fill empty cells with the possible notes

Give up button

Shows the answer to the puzzle

Timer

Able to stop, hide and reset timer. Function found in Options tab
